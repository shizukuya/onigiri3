# Assets フォルダ

このフォルダには、ゲームで使用するキャラクター画像を配置してください。

## 必要な画像ファイル

以下の9つのキャラクター画像が必要です：

- `char1.png` - キャラクター1
- `char2.png` - キャラクター2
- `char3.png` - キャラクター3
- `char4.png` - キャラクター4
- `char5.png` - キャラクター5
- `char6.png` - キャラクター6
- `char7.png` - キャラクター7
- `char8.png` - キャラクター8
- `char9.png` - キャラクター9

## 画像の仕様

- **形式**: PNG (透過背景推奨)
- **サイズ**: 正方形（推奨: 256x256px または 512x512px）
- **背景**: 透過または白背景
- **内容**: それぞれ異なるキャラクターまたはアイコン

## Expoアプリで必要な追加画像

以下のファイルも必要です（Expoのデフォルト画像を使用する場合は不要）：

- `icon.png` - アプリアイコン (1024x1024px)
- `splash.png` - スプラッシュ画面 (1242x2436px)
- `adaptive-icon.png` - Androidアダプティブアイコン (1024x1024px)
- `favicon.png` - Webファビコン (48x48px)

## プレースホルダー画像の作成方法

開発中にプレースホルダー画像を使用する場合は、以下のような方法で簡単に作成できます：

1. オンライン画像生成ツール（例: placeholder.com）を使用
2. グラフィックソフトで簡単なアイコンを作成
3. 無料のアイコンセット（例: Flaticon, Freepik）をダウンロード

**重要**: 実際のアプリをリリースする場合は、著作権に注意してください。
